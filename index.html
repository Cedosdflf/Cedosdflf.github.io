UTS Pemrograman Web

Buatlah 3 halaman HTML: Halaman Utama Halaman Formulir Halaman Tabel

Deskripsi singkat website Judul halaman yang sesuai Menu navigasi untuk berpindah ke halaman lain (Form dan Tabel) Deskripsi singkat tentang website

  • Aturan Desain Layout

    1. Buatlah setiap halaman dengan layout 3 baris (header, body, footer).
    2. Pada baris body terdapat (sidebar, konten utama)
    3. Header harus berisi judul website + logo.
    4. Konten utama berisi beberapa paragraf teks.
    5. Sidebar berisi daftar link ke halaman eksternal (misalnya link ke halaman form atau tabel).
    6. Footer berisi informasi hak cipta dan tahun pembuatan website.

  • Aturan CSS

    1. Setiap halaman file html memiliki 1 file css
    2. Gunakan minimal 5 jenis selektor dalam mendesain halaman

  • Bonus

    1. Tabel memiliki 1 data tambahan jika halaman diakses dari halaman form dengan melakukan submit data (ada javascript)
    2. Mengganti penggunaan data mahasiswa dengan data lain yang lebih panjang, misal: film
    3. Membuat tampilan yang menarik lebih bagus dari aturan layout
